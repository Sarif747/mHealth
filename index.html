Methodology - DSMA for Resilient Georgia

This study employs a mixed-methods approach combining synthetic ACE data analysis with stakeholder feedback to develop and evaluate the DSMA's AI-powered dashboard system for optimizing trauma-informed resource allocation and tracking intervention effectiveness.

Key Components

  • Data Collection: Synthetic RG datasets mirroring real-world distributions
  • Tools: Python, Dash, Plotly, SQLAlchemy, LangChain and OpenAI API
  • Analysis: Quantitative and Qualitative metrics on training data + survey reports
  • Output: Interactive dashboards with heatmaps, training effectiveness tracking and AI powered chat box for data insights
  • Evaluation: Measures both short-term and long-term systemic impact to remove intergenerational trauma
